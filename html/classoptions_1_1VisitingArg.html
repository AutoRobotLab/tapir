Detailed Description

template<typename ArgType, typename ValueType, typename OptionsType>
class options::VisitingArg< ArgType, ValueType, OptionsType >

A templated class representing a single command line alias.

This is basically just a wrapper for a TCLAP::Arg which uses the TCLAP::Visitor feature to offer an event-based way of handling how the arguments are read.

As such, ArgType must be a subclass of TCLAP::Arg in order to work.
