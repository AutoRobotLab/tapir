Detailed Description

A basic implementation of the SearchStrategy interface which should be sufficient for most purposes.

This is done by wrapping a StepGeneratorFactory, which will create StepGenerator instances, which in turn will create steps for the sequence, one step at a time.

This class also wraps a heuristic, which will be applied to evaluate non-terminal states that occur as the last entry in a sequence.
