std::vector< std::string > shared::split_function ( std::string  text)

Splits the given string as a function, i.e.

having the form functionName(arg0, arg1, ...).

This function does basic text parsing; nested brackets are assumed to be part of the arguments if they occur, and these can later be parsed in turn if the parser expects to see nested functions.
