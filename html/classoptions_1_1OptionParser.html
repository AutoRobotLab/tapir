Detailed Description

The core class for parsing options.

This class internally stores a TCLAP::CmdLine object to handle parsing of command lines, but also stores an unordered_map-based data structure to map INI section headings and option names to the associated options.

Note that each option is associated with a pointer-to-class-member object, which allows the option parser to be constructed once and used on different configuration files to load different options of the same kind.
