Detailed Description

Provides an implementation of the observation mapping interface that is designed for an enumerated set of observations - this set should generally be relatively small, and they should occur relatively non-sparsely.

Using this implementation requires that the observations implement DiscretizedPoint, which allows the mapping to determine the enumeration number associated with any observation. Each mapping simply stores its entries in an array, where the array index is the enumeration number.
