Detailed Description

An abstract base class that defines a mapping of the actions of a belief to (belief, action) edges in the belief tree; these edges are represented by the ActionMappingEntry class.

Each of these edges must also store the statistics for that edge - most notably, the visit count and estimated Q-value.

virtual long solver::ActionMapping::getNumberOfVisitedEntries ( ) const
pure virtual

Returns the number of entries in this mapping with a nonzero visit count.

This is different to the the number of child nodes, because sometimes there will be child nodes with zero visit counts due to deleted entries, and sometimes there will be nonzero visit counts without an associated action node due to initialization with nonzero values.
