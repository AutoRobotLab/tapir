Detailed Description

Represents a single history sequence.

The sequence owns its entries, which are stored in a vector of unique_ptr<HistoryEntry>.

The sequence also keeps track of the first index and last index for entries that have been affected by changes, as well as the logical disjunction (or) of all changes that affect the entries in the sequence.
