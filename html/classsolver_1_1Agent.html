Detailed Description

Represents an agent embedded within an environment.

The agent is tied to a solver, which in turn has an associated model of the environment and uses the ABT algorithm to find a good policy for the agent to take.

The agent represents its current state by associating it with a node in the solver's belief tree.

Note that multiple agents can use the same solver, as the belief tree representation is not tied to the state of a specific agent.
