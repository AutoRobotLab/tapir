Detailed Description

template<typename ValueType, typename OptionsType>
class options::Option< ValueType, OptionsType >

A templated class representing a single option.

As with the base option, the option is identified by its section and name. This class also stores a pointer-to-class member so that it knows its associated option, and can optionally store a default value.
