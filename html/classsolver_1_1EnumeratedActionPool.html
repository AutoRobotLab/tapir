Detailed Description

Defines an action pool that uses an enumerated set of actions.

This class keeps a vector of all of the possible actions, which it then uses in order to conveniently implement the abstract class DiscretizedActionPool.

std::vector< long > solver::EnumeratedActionPool::createBinSequence ( BeliefNode *  node)
overridevirtual

This provides a default implementation of createBinSequence(), which works by simply creating a vector of all the bins [0, 1, ..., getNumberOfBins()-1], and shuffling that vector.

This means that the actions will be taken in a random order, but the order is determined at the time this mapping is created. Taking this approach means the same interface can be used to make a deterministic, prioritized list of actions instead..

Implements solver::DiscretizedActionPool.
